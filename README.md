# Pixsols Protocol

Operational protocol layers enforcing the Pixsols Constitution.

## Canonical Documents

Pixsols is defined by three public, interlocking documents. Together they describe
the constitutional, operational, and explanatory layers of the system.

- **Pixsols Constitution**  
  https://github.com/aurionixlabs/pixsols-constitution  
  Binding invariants and non-negotiable constraints governing Pixsols and any system operating under it.

- **Pixsols Mandatory Protocol Pathways**  
  https://github.com/aurionixlabs/pixsols-protocol/tree/main/pathways  
  Operational enforcement pathways defining when Pixsols transitions from optional infrastructure to mandatory protocol.

- **Pixsols Whitepaper**  
  https://github.com/aurionixlabs/pixsols-whitepaper  
  Public explanation of Pixsols as a legitimacy protocol for intelligence-bearing systems.
